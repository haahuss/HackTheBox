Find Open Ports:
```
sudo nmap TARGET -p- -T5 -oA listopenports -Pn -n --disable-arp-ping

PORT   STATE SERVICE
22/tcp open  ssh
80/tcp open  http

```

List of open ports:
```
cat listopenports.nmap | grep open | grep tcp | cut -d "/" -f1 | tr "\n" ","


```

```
sudo nmap TARGET -p PORTS -A -oA fullscan -Pn -n --disable-arp-ping
```
